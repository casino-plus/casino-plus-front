# Casino+ (フロントエンド)

ようこそ カジノ+ のフロントエンド開発へ
こちらは Flutter を使ったクライアントアプリです
使用に当たっては事前に Flutter 実行環境をご用意ください

<br>

## 使い方

### 1. Chrome でアプリを動かす

```
$ futter run -d chrome
```

### 2. iOS, Android など接続された全てのプラットフォームで動かす

事前にそれぞれの起動が必要です

```
$ futter run -d all
```
